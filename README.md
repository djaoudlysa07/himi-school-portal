HIMI School Management Portal
Institution: Higher International Management Institute
Academic Year: 2025–2026

Project Overview:
HIMI School Portal is a full-stack web application designed and developed 
to digitise the academic management process at the Higher International 
Management Institute, situated in Tizi Ouzou, Algeria. The system provides a centralised platform accessible 
to three categories of users: students, teachers, and administrator,
each with a dedicated interface tailored to their roles and permissions.

Technical Stack:
- Backend:  Python 3, Flask (web framework), SQLite (database)
- Frontend: HTML5, CSS3, JavaScript
- Fonts:    Archivo Black (headings), Figtree (body)
- Hosting:  Render.com (live deployment), GitHub (version control)

Features by Roles:

  Student Portal:
  - Consult grades and academic average per subject
  - View weekly timetable for their class group
  - Track pending, overdue, and completed assignments
  - Access lessons published by teachers
  - View personal attendance record

  Teacher Portal:
  - Publish and manage lessons per subject and class
  - Create and assign homework to students
  - Record student attendance (present, absent, late)
  - Add grades for individual students
  - Post announcements to the whole school

  Administrator Portal:
  - Full access to all teacher features
  - Manage the weekly timetable for all classes
  - Create, modify, and delete user accounts
  - Delete announcements, grades, and assignments
  - Monitor all students and teaching staff
